# ether-ai-core

Contratos neutrales para capacidades de GenAI dentro del ecosistema Ether. Este módulo concentra tipos y APIs compartidas sin acoplarse a un proveedor concreto.

## Maven Dependency

```xml
<dependency>
    <groupId>dev.rafex.ether.ai</groupId>
    <artifactId>ether-ai-core</artifactId>
    <version>8.0.0-SNAPSHOT</version>
</dependency>
```
